About me

I build the things
designers hand off.

I spent years as a UI developer at a digital agency, building interfaces for ecommerce clients — product pages, checkout flows, landing pages, and everything in between. I've implemented enough designs to know what works in the browser, what developers actually need, and where design decisions create problems down the line.

Along the way I taught myself Figma and Sketch, so I can now handle both sides. I'm stepping into freelance for the first time — offering UI development, redesign, and design-to-code work for clients who want one person who understands the full picture.

How I can help you

01

Frontend Implementation

My core skill. I translate designs into clean, responsive HTML and CSS that actually match the mockup — no guesswork, no back-and-forth. Ideal for teams who need a developer who can read a Figma or Sketch file fluently.

  • HTML & CSS build-out
  • Responsive & mobile-first layouts
  • Figma & Sketch to code
  • UI polish & pixel refinement
  • Ecommerce page builds
02

Redesign & Refresh

Got a page that's not converting, or an interface that just feels dated? I'll review what's not working and rebuild it — then implement it myself if needed, no handoff required.

  • Usability & layout review
  • Conversion-focused redesign
  • Visual refresh & modernization
  • Mobile-first optimization
  • Ecommerce page redesign
03

UI Design

Self-taught in Figma and Sketch, I can put together wireframes and high-fidelity mockups for smaller scopes — particularly useful when you need design and build handled by the same person.

  • Wireframing & layout
  • High-fidelity mockups
  • Figma & Sketch files
  • Mobile-first design
  • Design-to-code delivery
